차체 유형별 분류

  • 세단(Sedan)
    • 일반적인 4도어 차량 (예: 현대 소나타, BMW 5시리즈)
  • 해치백(Hatchback)
    • 트렁크와 뒷유리가 연결된 형태 (예: 폭스바겐 골프, 현대 i30)
  • 쿠페(Coupe)
    • 2도어 스포츠형 디자인 (예: 포르쉐 911, 벤츠 C클래스 쿠페)
  • 컨버터블(Convertible)
    • 지붕이 열리는 오픈카 (예: 마즈다 MX-5, BMW Z4)
  • suv
    • 소형 SUV : 기아 셀토스, 현대 코나
    중형 SUV : 현대 싼타페, 도요타 RAV4
    대형 SUV : 제네시스 GV80, 캐딜락 에스컬레이드
  • 크로스 오버
    • 승용차와 SUV의 중간 형태 (예: 닛산 캐시카이, 볼보 XC40)
  • mpv
    • 다양한 용도로 활용 (예: 기아 카니발, 혼다 오딧세이)
  • 미니밴
    • 가족용으로 널리 사용됨 (예: 토요타 시에나)
  • 픽업트럭
    • 짐칸의 덮개가 없고 측면이 차체와 이어진 소형 트럭의 일종 (예: 포드 F-150, 램 1500)
  • 트럭
    • 대형 화물 운송 차량 (예: 볼보 FH, 현대 엑시언트)
  • 스포츠카
    • 경량 고성능 자동차 (예: 닛산 GT-R, 포르쉐 911)
  • 슈퍼카
    • 최고급 고출력 차량 (예: 람보르기니 아벤타도르, 페라리 SF90)
  • 하이퍼카
    • 극한 성능을 지닌 초고가 모델 (예: 부가티 시론, 코닉세그 레게라)

용도별 분류

  • 일반 승용차 : 일상용 (세단, 해치백 등)
  • 스포츠카 & 레이싱카 : 고속 주행, 트랙 주행 목적
  • 오프로드 차량 : 험로 주행 가능 (예: 지프 랭글러, 랜드로버 디펜더)
  • 상업용 차량 : 물류, 운송 목적 (트럭, 밴)
  • 군용 & 특수 차량 : 군사 작전, 소방차, 경찰차 등

연료 타입별 분류

  • 내연기관 (ICE, Internal Combustion Engine)
    • 가솔린 (휘발유)
    디젤
    LPG
  • 전동화 차량
    • 하이브리드 (HEV) : 가솔린 + 전기
    플러그인 하이브리드 (PHEV) : 전기 충전 가능
    전기차 (EV)
    수소차 (FCEV)

구동 방식별 분류

  • 전륜구동 (FWD, Front-Wheel Drive) : 앞바퀴로 구동
  • 후륜구동 (RWD, Rear-Wheel Drive) : 뒷바퀴로 구동
  • 사륜구동 (AWD, 4WD, All-Wheel Drive) : 네 바퀴로 구동
